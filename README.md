Simplifies the descriptions of all POLTERWORX items by:<br>
- Rewording cards' descriptions for consistency
- Removing all formatting in card names
- Removing all size changes above 1
- Removing all text motions
- Replacing most colors/gradients with vanilla colors/dark edition
